# Markit Language Support

Language support for the [Markit](https://github.com/earlytexts/markit) markup language for early texts.

## Basic Features

- Syntax highlighting
- Live error reporting
- Folding ranges
- Auto-closing pairs and snippets

## Commands

- `Alt+Shift+F`: format document
- `Cmd/Ctrl+Shift+V`: live HTML preview to the side
- `Cmd/Ctrl+Shift+P`, then search for "Markit" to see compilation commands:
  - Compile to HTML
  - Compile to JSON
  - Compile to Text
